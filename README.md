# 🍕 Chez Anas Pizza – Site Vitrine

Site web vitrine pour la pizzeria **Chez Anas Pizza** à Montereau-Fault-Yonne.

## 📁 Structure du projet

```
chez-anas-pizza/
├── index.html       # Page d'accueil
├── menu.html        # Page menu avec filtres
├── contact.html     # Page contact + formulaire
├── style.css        # Styles CSS (thème noir/jaune/rouge)
├── script.js        # JavaScript (burger menu, filtres, formulaire)
└── README.md
```

## 🎨 Identité visuelle

- **Couleurs** : Noir `#111`, Jaune `#f5c800`, Rouge `#e01a1a`, Vert `#2a8a3e`
- **Typographie** : Pacifico (titres) + Nunito (corps)
- Inspiré du logo et de la devanture Chez Anas Pizza

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Hero animé, offre 1+1, cards pizzas vedettes, infos pratiques |
| `menu.html` | Menu complet avec filtres par catégorie et 3 tailles de pizza |
| `contact.html` | Coordonnées, horaires, formulaire validé, carte Google Maps |

## 🚀 Publier sur GitHub Pages

1. Créer un dépôt GitHub (ex. `chez-anas-pizza`)
2. Uploader tous les fichiers à la racine
3. Dans **Settings → Pages** → Source : `main` / `root`
4. Votre site sera disponible sur : `https://votre-username.github.io/chez-anas-pizza/`
